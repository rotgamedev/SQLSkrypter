Official User Manual

Operations, limitations, and architectural overview (v2.0.0).

Standard vs. PRO Edition Comparison

To ensure high performance and sustainability, SQLSkrypter uses a tiered feature model.

  • - See Licensing section for exact feature limitations.

Setting Up Your Environments

SQLSkrypter allows you to organize your servers into logical groups.

  • Standard Edition: Create up to 3 hierarchical environments.
  • Visual Distinction: Custom hex colors are allowed globally.

Script Execution & Modes

The Execute button runs the current script on all checked databases.

  • Sequential Execution: Operates one after another.
  • Dry Run (PRO): Wraps query in a simulated Transaction rollback.

Professional Productivity Features (PRO)

Advanced lifecycle tools for DBAs.

  • Environment Dashboard (Chips): Isolate Dev/Prod branches instantly.
  • Configuration Mobility: Save complex XML structures with AES encryption.
  • Tree Sessions: Remember exact checkboxes.

Security & Privacy

100% Offline application design.

  • No internet telemetry is ever reported.
  • Zero dependencies on cloud validation nodes.
  • DPAPI storage encryption for offline credential safety.
